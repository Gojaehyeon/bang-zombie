# 빵! 좀비 (Bang Zombie)

맥북 웹캠으로 내 손을 권총으로 인식해서, "빵!" 제스처로 좀비를 쏘는 브라우저 기반 FPS.

> 설치 제로. 맥북 + Chrome + 웹캠만 있으면 바로 플레이.

## 상태
🚧 **기획 단계 완료.** 구현은 새 세션에서 시작.

## 문서
- **[기획서](docs/기획서.md)** — 전체 게임 디자인, 제스처 인식 알고리즘, 마일스톤
- **[새 세션 프롬프트](docs/프롬프트.md)** — 구현 시작할 때 복붙할 프롬프트

## 핵심 아이디어
1. 웹캠으로 손 랜드마크 21개 추출 (MediaPipe Hands)
2. 검지 뻗고 엄지 위, 나머지 접음 = **권총 포즈**
3. 엄지 꺾기 or 검지 튕김 = **발사**
4. 검지 끝 = **크로스헤어**
5. 좀비 처치 → 점수 → 웨이브

## 스택 (예정)
Vite + TypeScript + HTML5 Canvas + MediaPipe Tasks Vision

## 시작하는 법 (구현 후)
```bash
npm install
npm run dev
# → http://localhost:5173 에서 카메라 권한 허용 → 플레이
```
